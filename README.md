# Smart City Traffic Analysis – Power BI

## Project Overview
An interactive Power BI dashboard developed to analyze real-world traffic volume data for smart city traffic management. The project identifies congestion patterns, peak hours, and long-term traffic trends using time-series analysis.

## Tools & Technologies
- Power BI
- DAX

## Key Features
- Star schema data model with Date dimension
- KPI cards and interactive slicers
- Heat maps for congestion analysis
- Outlier detection using IQR method
- 7-day and 30-day moving average forecasting
- Forecast accuracy evaluated using MAE (~267 vehicles)

## Key Insights
- Peak traffic during 10–12 AM and 7–8 PM
- Junction 1 identified as the main congestion hotspot
- Weekday traffic higher than weekends
- Medium traffic category dominates
- Forecast errors increase during sudden traffic spikes
